# **App Name**: CliqueyTalk

## Core Features:

- Voice Channel Creation: Allow users to create private voice channels with customizable settings (volume, muting, etc.) for up to 50 participants.
- User Authentication and Management: Secure user accounts with Discord/Twitch authentication. This also includes profiles and settings.
- Discord and Twitch Bot Integration: Integrate the app with the existing Discord and Twitch bots for user authentication and channel management.
- Real-time Voice Communication: Enable real-time, low-latency voice communication within channels.
- Channel Configuration AI Tool: AI tool that uses user descriptions of the kind of group they want to form to suggest initial configuration settings for the voice channel, such as volume levels and muting settings.
- WebRTC Audio Management: Individual level audio customizations between speakers.
- Admin Section: Admin interface to drag and drop users between channels, close channels, and ban users, similar to Twitch's mod view. Includes a dedicated voice and text chat channel for communication with moderators.
- User Status Overlay: A small browser popout showing a scrolling list of active users from Twitch and Discord, with their names and statuses (sleeping, muted, listening, playing game). Includes a toggle to turn the background green for chroma keying.

## Style Guidelines:

- Primary color: A vibrant purple (#A020F0) to reflect energy and community.
- Background color: A very light, desaturated purple (#F0E5F5).
- Accent color: An analogous blue (#20A0F0) to complement the purple.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text. 'Space Grotesk' is a sans-serif, while 'Inter' is also a sans-serif.
- Clean and intuitive layout that prioritizes ease of navigation.
- Custom icons to represent voice chat features and channel settings.
- Subtle transitions and feedback animations.
